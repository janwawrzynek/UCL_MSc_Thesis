# UCL_MSc_Thesis
Here is the python codebase for my UCL MSc in Physics thesis. To run the code , all of the files need to be in the same folder. Then navigate to the parent folder which contains this UCL_MSc_Thesis and in the terminal type: python-m  UCL_MSc_Thesis.simulation to run the code. To change the scenario, vary the task string as described inside the simulation file. 
The full project report, including physical theory deep-dive as well as OOP design documentation.
